Markdown Editor and Previewer  with Create-react-app and marked npm package

Follow below steps to run the app :
1. clone the repo `git clone`
2. cd `markdown-editor-app`
3. run `npm install`
4. run `npm start`
5. Live at the `localhost:3000`
